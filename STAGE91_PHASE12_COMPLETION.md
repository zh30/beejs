## ICU 限制说明

### 问题描述
在大量迭代计算时（> 1000 次），beejs 运行时遇到：
```
TypeError: Internal error. Icu error.
```

### 影响
- 无法进行大规模基准测试
- 无法验证高迭代性能指标

### 解决方案
1. **功能性验证**: 通过基本功能测试验证运行时正常
2. **轻量级测试**: 使用小规模测试验证性能趋势
3. **特性验证**: 验证运行时特性和优化机制
4. **外部测试**: 使用 Rust 基准测试验证性能
