### 核心算法

#### 1. 异常检测算法
```rust
// Z-score 检测
let z_score = (value - mean) / std_dev;
if z_score.abs() > threshold {
    // 检测到异常
}

// Level Shift 检测
let shift = (second_half_mean - first_half_mean) / first_half_mean;
if shift.abs() > level_shift_threshold {
    // 检测到水平位移
}
```

#### 2. 趋势分析算法
```rust
// 线性回归计算斜率
let slope = (n * sum_xy - sum_x * sum_y) / (n * sum_x2 - sum_x * sum_x);

// R-squared 计算
let r_squared = 1.0 - (ss_res / ss_tot);
```

#### 3. 故障预测算法
```rust
// 多因子评分
let mut probability = 0.0;
if has_anomaly { probability += 0.4; }
match trend_direction {
    Upward => probability += 0.4,
    Downward => probability += 0.2,
}
if growth_rate > 0.3 { probability += 0.4; }
```
